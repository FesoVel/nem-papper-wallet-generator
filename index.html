How does this work?

This is a browser based NEM papper wallet generator built on JavaScript. All of the code runs on your browser. It generates a new NEM wallet that you can use to transfer funds to. To generate new keys and addresses it uses the functionality of the NEM SDK. It does not store any wallet information and nothing ever leaves your browser. This is an open source project and is hosted on GitHub.

Safety tips

Before using this wallet generator please make sure to disable your browser extensions. You can turn off your internet connection before using the generator, it will work eather way.

For maximum safety download the code from GitHub and run it on your local computer that is not connected to the internet.

If you are more trusting, you should still think about running the generator from an incognito window.

Remember that ANYONE who has access to your private key can spend your XEM. Keep your wallet SAFE!

For Your Safety

Please make sure you are using a safe browser and you understand that the private key and the addres will be revealed on your screen before you proceed. Keep your wallet safe!
